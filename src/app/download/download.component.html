Securely share your files without server

ShareTC is a service allowing you to securely share file with someone else. It works without any server, your browser will act as the sender, so keep your browser open while sending. The service do not use any server thanks to the RTC peer-to-peer technology. You'll need to communicate the generated key to your sender to be able to download file.

Sender configuration Enter the sender configuration and click connect Contains your generated configuration that you need to share with sender

To connect copy sender generated configuration and click connect. This will generate your configuration that you should share with sender.

After that you'll automatically download all sender file named as file.dat.
